Skill scatter

Each point = one skill. X = mean status gap. Y = mean log-hazard. Size ∝ frequency. Click a matrix cell to highlight skills in that cell.

How to read

Rows = log-hazard tercile (High / Mid / Low).

Columns = status gap zone:
T1 Target ranks below source (downward gap, G < 0) · T2 Peer / lateral (G ≈ 0) · T3 Target ranks above source (upward gap, G > 0)

lp = log-hazard of diffusion. Higher lp = higher probability that the skill is adopted (or abandoned). Lower lp = lower probability — the status gap acts as a barrier.

Green = facilitated   Red = blocked

Source → Target = most frequent real observed occupational pair for that skill in the cell (O*NET 2015–2024)

Fixed effects selectable above (Source + Skill FE or Target + Skill FE). Status gap = PC1(target − source), composite of log wage, log education, and cognitive content. Skill profile distance = cosine distance between occupation skill-specialization profiles (O*NET RCA ≥ 1). Full: all dyads in deciles D2–D9 of the distance distribution. Trimmed: restricts to D4–D6 — pairs with moderate profile dissimilarity, excluding both very similar and very different occupation pairs.
