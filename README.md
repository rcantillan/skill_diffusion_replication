# A Matthew Effect in Occupational Skill Content

### Divergent Skill Diffusion Reinforces Occupational Stratification

*Manuscript under review*

---

## Overview

This repository contains all code and derived data necessary to reproduce the analyses, figures, and supplementary materials reported in the paper. The project documents asymmetric directional flows of skill requirements across the occupational hierarchy: socio-cognitive skills concentrate upward while sensory-physical skills accumulate downward — and critically, both adoption and abandonment reinforce this divergence rather than attenuate it. This bilateral pattern constitutes a structural Matthew effect in occupational skill content.

The core analytical contribution is a gravity model with a bilateral, directional specification applied to approximately 40 million directed dyadic observations across 741 occupations and 160 skill requirements from O\*NET (2015–2024), estimated via `feglm` with a complementary log-log link and three-way clustered standard errors.

## Data

**Primary source:** O\*NET database, versions 20.0–29.0 (2015–2024).
Download from: [https://www.onetcenter.org/database.html](https://www.onetcenter.org/database.html)

Place raw O\*NET files under `data/raw/onet/db_XX_X_text/` following the standard O\*NET folder naming convention. The risk-set scripts (`01a`, `01b`) use BLS Occupational Employment and Wage Statistics (OEWS) data for wage imputation; download from [https://www.bls.gov/oes/](https://www.bls.gov/oes/) and place under `data/raw/bls/`.

Derived datasets (`data/derived/`) are not tracked due to size (~40M observations per flow). They are fully reproducible from raw data using the pipeline scripts in order.

The status score file `output/tables/main/occ_status_scores.csv` is tracked and contains the PC1 composite index (wage, education, cognitive content) for all 741 occupations.

## Reproducing the Analysis

### Requirements

- R ≥ 4.3
- Packages: `data.table`, `fixest`, `ggplot2`, `ggsci`, `patchwork`, `cowplot`, `jsonlite`, `igraph`, `ggraph`
- RAM: ≥ 16 GB recommended (adoption risk set: ~21M observations; abandonment: ~18M)
- Swap: configure ≥ 16 GB if RAM is limited

### Setup

Create `R/99_paths_local.R` with your local data root:

```r
BASE <- "~/Documents/Papers/skill_diffusion"
```

### Main pipeline

```bash
Rscript R/01a_risk_set_adoption.R
Rscript R/01b_risk_set_abandonment.R
Rscript R/02a_enrich_adoption.R
Rscript R/02b_enrich_abandonment.R
Rscript R/03_nestedness.R          # ~2–3 hours; checkpoint/resume supported
Rscript R/03b_nestedness_merge.R
Rscript R/03c_nestedness_merge_ab.R
Rscript R/04_status_pca.R
Rscript R/05a_gravity_adoption.R
Rscript R/05b_gravity_abandonment.R
Rscript R/06_projections.R         # Generates Fig. 3 and projection tables
Rscript R/07_fig1_descriptive.R
Rscript R/08_fig2_regression.R
```

For computationally intensive steps, redirect output to `logs/`:

### Gravity model specification

The main models are estimated with `feglm(..., family = binomial("cloglog"))` from the `fixest` package. Panel A (source + skill fixed effects) is the preferred specification. Key directional predictors:

- `pc1_up` = max(0, G_ij) where G_ij = status_target − status_source
- `pc1_down` = min(0, G_ij)
- `up_dummy` = I[G_ij > 0]

All models use three-way clustered standard errors (source × target × skill).

```bash
cp R/SI/explorer/triads_explorer.json docs/triads_explorer.json
```

## Key Results

| Flow | Skill domain | Status gap coefficient | Direction |
|------|---|---|---|
| Adoption | Socio-cognitive | β↑ > 0 | Upward facilitated |
| Adoption | Sensory-physical | β↑ < 0 | Upward blocked |
| Abandonment | Socio-cognitive | β↑ < 0 | Shedding facilitated downward |
| Abandonment | Sensory-physical | β↑ > 0 | Shedding blocked downward |

The directional model recovers 50–115% of the observed Q5–Q1 gradients across flows and skill types. Occupational status index: PC1 explains 75.0% of variance in wage × education × cognitive content.
